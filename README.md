# wikiapp

This app uses Wikipedia API to fetch the relevant data and display the random articles, featured images, category list on Landing Screen.
Room/SQLite database to Store the data & support offline reading. 

Tech Stacks :
Kotlin
MVVM architecture

Libraries Used : 

Room Database
Retrofit
Picasso
Android Lifecycle
